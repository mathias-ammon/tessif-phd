.. _Examples:

examples
========

.. automodule:: tessif.examples

.. toctree::
   :maxdepth: 2
   :titlesonly:

   examples/application
   examples/data
   examples/transformation


- :mod:`~tessif.examples.application` is a :mod:`tessif` subpackage for providing:

   - Examples showing the 'whole picture' of using tessif.
   - Code references on conducted research using tessif
   - Reusable code for conducting your own research

   It serves as main reference point for experienced :mod:`tessif` users.


- :mod:`~tessif.examples.data` is a :mod:`tessif` subpackage for providing:

   - Examples showing data input handling
   - Specificatons on the expected format for providing externalised data

   It serves as main reference point for data input related questions as well as externalized data format support.


- :mod:`~tessif.examples.transformation` is a :mod:`tessif` subpackage for providing:

   - Utility wrappers for creating (transformed) energy systems ready to compare
   - Examplary code and explanations on less straight forward transformations

   For a basic expalantory example see User's Guide section :ref:`Transformation`
